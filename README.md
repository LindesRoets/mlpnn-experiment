# Multilayer Perceptron Cloud Application


<br/>
**Download source**

```git
git clone git@github.com:LindesRoets/mlpnn-experiment.git
```

<br/>
**Compile source code**
```maven
mvn compile
```

<br/>
**Dowload and Transform Data Sets in offline mode**
```maven
mvn test -Dtest=DownloadDataSetTest
```

<br/>
**Run application**
```maven
mvn spring-boot:run
```

<br/>
**Create java docs**
```maven
mvn javadoc:javadoc
```
